How to create the token (one-time, ~2 min)

GitHub → your avatar → Settings → Developer settings (bottom of left sidebar) → Personal access tokens → Fine-grained tokens → Generate new token.

Name it DDI Admin. Set expiration (1 year is fine — you'll paste a fresh one when it expires). Under Repository access choose Only select repositories → ddi-cloud. Under Permissions → Repository permissions set Contents to Read and write. Generate, copy the github_pat_… string, paste it above. It's stored only in this browser.
